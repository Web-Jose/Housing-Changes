Proxy Server for Event and Weather Signage

The RSS feeds for the Event and Weather Signage had some issues with cross-origin requests, so I created a proxy server to fix the issue. I am using the proxy server to get the data from the RSS feeds and then I am using the data to create the signage. It updates on request and it is working well.
