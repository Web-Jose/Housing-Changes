Deactived Plugin Automatic Responsive Tables

I deactivated the plugin Automatic Responsive Tables as it was causing issues with the tables on the website. When in use it would cause tables on tablet and mobile to be bright red and blue and would not display the table correctly. After it was deactivated tables were displaying correctly again.
