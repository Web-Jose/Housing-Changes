Jose's Weekly Web Design Updates

Week 64

8/5/2024- 8/9/2024

Summary

This week I have been working on making the website more user friendly to new residents. I have been adding floor plans to the nav menu, changing links on the floor plans page to open in a new tab, creating new journey maps for pages that new residents will look for, remaking the news and events widget to be more dynamic, and creating an event template to make it easier to add information to the event pages. I have also been training the new web designer and adding floor plans to the homepage. This week has been very productive as I prepare to leave the position.

Maintenance and Updates

Added Floor Plans to the Nav Menu

In my effort to make the website more user-friendly to new residents, I added the floor plans to the nav menu.

Changed Links on Floor Plans Page to Open in a New Tab for Better User Experience

I changed the links on the floor plans page to open in a new tab for better user experience. This way, the user can easily navigate back to the floor plans page without losing their place.

Created New Journey Maps for Pages that New Residents will Look For

In Figma I have been creating journey maps for pages that new residents will look for. Journey maps are a visual representation of the process that a person goes through to accomplish a goal. I am trying to make it as easy as possible for new residents to find the information that they need.

Remade the News and Events Widget to be More dynamic

This widget used to display events even if they were in the past. I have remade the widget to only display events that are in the future. This way, the widget is more dynamic and will always show relevant information to the user. I am working on adding interactivity to the widget so that users can click on the events to get more information.

Created an Event Template to Make it Easier to add information to the event pages

I have created an event template to make it easier to add information to the event pages. This way, all the event pages will have a consistent look and feel. I have also been working on adding interactivity to the event pages so that users can click on the events to get more information.

Training the New Web Designer

I have been training the new web designer. I have been preparing them for when I leave. I try and give task and check in on them to see how they are doing. I have been trying to make sure that they have all the information that they need to be successful. I think he is fitting in well and is learning quickly.

Added Floor Plans to the Homepage

I added a button to the homepage that links to the floor plans page. This way, new residents can easily find the floor plans and see what options are available to them.
